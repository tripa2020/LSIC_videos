# plans/

Paired plan documents for **major efforts** in the LSIC pipeline (a new module set, a
multi-part build, or a new external surface such as cloud/API). See CLAUDE.md §1.

Each effort produces two files sharing one `<NAME>` (UPPER_SNAKE):

- `<NAME>_PLAN.md` — **durable**, the living source of truth, kept synced to the code.
- `<NAME>_DESIGN_RATIONALE.md` — **frozen** provenance, written once at planning time and
  never updated afterward.

Generated by `/deep-plan` (Phase 4) from `.claude/skills/deep-plan/reference/plan-template.md`.
Anything smaller than a major effort just updates the master `PLAN.md` at repo root.
